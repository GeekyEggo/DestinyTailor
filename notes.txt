/Platform/Destiny/{platform}/Account/{accountId}/Character/Inventory/Summary/?definitions=true
    Loads information about the items on a character
    
/Platform/Destiny/{platform}/Account/{accountId}/Character/{characterId}/Inventory/{itemId}/?definitions=true
    Loads information about an item
    [.data.item.stats] Contains the current equipped values within `.value` with talents applied!
    [.data.talentNodes] The talent nodes available to that item
        [.isActivated] Determines if the node is active
        
    [.data.item.nodes] Contains all of the nodes, including the activated ones, and their node hash
    [.data.item.statsOnNodes.{nodeHash}.currentNodeStats.[1/2]] The default value for items with no nodes.
        Also includes the increment value.
